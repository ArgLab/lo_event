lo-event Examples

These demos show how to use lo-event in a browser. Run them with: 

npm run browser

This starts a Parcel dev server that bundles the TypeScript sources on the fly.

  • Browser Events — Captures keystroke, mouse, clipboard, and other DOM events using subscribeToEvents. Events are displayed on-page and logged to the browser console via consoleLogger. Also demonstrates metadata collection (browser info, Chrome identity, local/session storage).
  • Redux Loop — Demonstrates the Redux logger integration. Events dispatched via lo_event.logEvent flow through Redux, so application state and event history live in a single store. Includes a button that emits a click event.
